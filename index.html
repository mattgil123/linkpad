// LINKPAD · Setup
one-time configuration · takes ~2 minutes
Step 01 — Create a GitHub Token

Go to github.com/settings/tokens/new, set the note to Linkpad, check only the gist scope, set expiration to No expiration, then click Generate token. Copy the token — it starts with ghp_.

Step 02 — Paste Your Token

Paste the token below. It will be stored only in this browser and used solely to read and write your private Gist.

// LINKPAD
idle
synced via github gist  ·  works on all devices
$linkpad --add-url
//
// no links saved paste a url above and hit save_
// Disconnect

This will remove your token from this browser. Your Gist data stays safe on GitHub. You can reconnect anytime with the same token.
